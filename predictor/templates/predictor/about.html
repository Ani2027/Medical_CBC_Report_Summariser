About This Project

This platform empowers users to upload their CBC blood reports and instantly receive AI-generated health insights. Our system extracts relevant markers from standard PDF reports, interprets them using clinical rules and a trained machine learning model, and presents results in plain, understandable language.

PDF Extraction

Automatically parses CBC values from standard lab report PDFs using PyMuPDF.

Rule-Based Interpretation

Each CBC parameter is explained using established clinical reference ranges.

ML Disease Prediction

A LightGBM classifier predicts the top 3 most likely conditions with confidence scores.

Personalized Advice

Diet recommendations and follow-up tests tailored to each predicted condition.

Developed By

Built by Aniket and Harshit — inspired by real-world healthcare needs and diagnostic workflows.
